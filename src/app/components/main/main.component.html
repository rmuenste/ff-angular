FeatFloWer

For Accurate and Robust FEM Simulations

High Accuracy

The FeatFloWer software package uses higher order FEM schemes to achieve accurate results. Using unstructured meshes with (dynamic) mesh adaptation can further increase simulation accuracy.

The core solver for the flow equations is based on a Newton-Multigrid approach.

Versatile Flow Models

Besides the generalized Newtonian fluid model, our software supports non-Newtonian power-law, Carreau and visco-elastic models

Multiphase models can be used to model different liquid-liquid configurations or liquid-solid flows.

Extensive Validation

In a continuous benchmarking effort we test the software with well-known benchmark configurations and flow problems.

Results and reference data are available for Newtonian and Non-Newtonian configurations, multi-phase problems and sedimentation problems with solid particles.
